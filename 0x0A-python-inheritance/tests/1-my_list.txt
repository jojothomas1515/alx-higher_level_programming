The ``inheritance`` module
==========================

Setup::
>>> MyList = __import__("1-my_list").MyList

Using ``print_sorted``
----------------------

Cheking if ``print_sorted`` works properly

::

>>> new_list = MyList((1, 3, 4, 6, 2, 5))
>>> new_list.print_sorted()
[1, 2, 3, 4, 5, 6]

>>> print(new_list)
[1, 3, 4, 6, 2, 5]


Testing unordered append

::

>>> new_list  = MyList()
>>> new_list.append(-3)
>>> new_list.append(3)
>>> new_list.append(1)
>>> new_list.append(-2)
>>> new_list.append(0)
>>> new_list.append(-1)
>>> new_list.print_sorted()
[-3, -2, -1, 0, 1, 3]
